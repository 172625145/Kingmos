# Kingmos
Kingmos is a os,include mini kernel, gui, driver, ide......
Kingmos 是一个嵌入式操作系统，包括微内核，图形窗口系统，设备管理及其各种驱动，KingBuilder（编译开发环境，IDE）。
Kingmos 是多进程，多线程系统。
Kingmos 最初在ARM 上完成开发完成。
Kingmos 第一版，开始于1998年。

为什么会开发Kingmos ？ 一个人做了一个激动人心的梦而已！
